# Traffic Light Controller using FSM in Verilog HDL

## Overview
This project implements a Traffic Light Controller using Finite State Machine (FSM).

## Tools Used
- Verilog HDL
- EDA Playground
- GTKWave

## FSM States
- S0 → Green
- S1 → Yellow
- S2 → Red
